## [0.1.0] - 2024-03-XX
### Added
- Initial release
- Worker pool implementation with configurable workers
- Task queue with thread-safe operations
- Retry mechanism for failed tasks
- Custom logging system
- Example implementations:

  - Email sender
  - Image downloader
